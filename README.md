# ✈️ OGY TRAVEL - Premium Management & Booking Portal

OGY TRAVEL is a luxury agency management and booking portal designed for unified ecosystem operations. It offers dual-role authorization matrices allowing administrators to manage vacation asset inventories and review macro metrics dashboards while regular clients can seamlessly explore curated packages, write reviews, and run secure checkout simulations.

---

## 🚀 Key Features

*   **Dual Identity Portal Verification:** Secure context switching dynamically separating layouts for administrators and standard users.
*   **Active Vacation Portfolio Catalogue:** Interactive destination matrix allowing automated grid rendering, custom listing filtering, and real-time administrative deletion operations ("eviction").
*   **Analytical Telemetry Insights Dashboard:** Graphical multi-color vector metric components driven by **Chart.js** detailing commercial volume statistics across active locations.
*   **Secured Settlement Gateway Simulation:** Multi-option sandbox transaction pipelines processing mock inputs for Local Credit/Debit Cards, FPX Automated Retail Banking Selection, or DuitNow/Touch 'n Go QR code execution tokens.
*   **Archival Spreadsheet Audit Records:** Dynamic system audit spreadsheet tables tracking verification identifiers, customer labels, dates, and currency totals.
*   **Print-Engine Receipt Synthesis Engine:** Custom CSS layout rules explicitly mapped for clean `@media print` physical print formats or flawless PDF saving workflows.
